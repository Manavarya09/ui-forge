# Playful Design Language

Rounded, colorful, fun design with bouncy animations, soft shapes, and friendly vibes.

## Features

- Bouncy animations with spring physics
- Soft, rounded shapes and corners
- Friendly color palette
- Confetti and decoration elements
- Warm, inviting atmosphere

## Usage

```bash
npx uiforge create my-app --style playful
```

## Design Tokens

- **Primary**: Pink (#f472b6)
- **Secondary**: Cyan (#22d3ee)
- **Accent**: Lime (#a3e635)
- **Background**: Warm cream (#fff7ed)

## Typography

- **Sans**: Nunito
- **Serif**: Fredoka
- **Mono**: Fira Code

## Animations

Uses cubic-bezier(0.68, -0.55, 0.265, 1.55) for bouncy spring effects on all interactions.
