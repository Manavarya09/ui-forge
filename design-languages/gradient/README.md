# Gradient Design Language

Bold gradient-heavy design with vibrant color transitions, modern depth, and luminous effects.

## Features

- Vibrant gradient backgrounds with mesh effects
- Orange to purple to cyan color transitions
- Glowing shadows and luminous effects
- Gradient border support
- Modern dark theme with high contrast

## Usage

```bash
npx uiforge create my-app --style gradient
```

## Design Tokens

- **Primary**: Orange (#f97316)
- **Secondary**: Purple (#8b5cf6)  
- **Accent**: Cyan (#06b6d4)
- **Background**: Deep navy (#0f0f23)

## Typography

- **Sans**: Outfit
- **Serif**: Playfair Display
- **Mono**: JetBrains Mono
